## 方法如下：

对于卸载 `rustc`, `rustup` 和 `cargo` 的命令为

```bash
rustup self uninstall
```
如果结束后提示 `error: could not remove 'cargo_home' directory`，我们可以手动删除cargo的目录就可以：
```bash
# MAC 环境cargo在用户目录下的.cargo文件夹
rm -rf ~/zhanghe/.cargo
```
