## RUST cargo 使用命令

```bash
cargo --version // 查看 cargo 的版本
cargo build // 构建项目并运行它
cargo build --release // 优化编辑
cargo run // 执行项目
cargo check // 快速检查您的代码以确保它编译但不生成可执行文件

```
